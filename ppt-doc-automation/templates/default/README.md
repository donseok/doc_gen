# Default Template Directory

이 디렉토리에 기본 PPT 템플릿 파일을 저장합니다.

## 사용 방법

1. PowerPoint에서 원하는 디자인의 템플릿을 만듭니다.
2. `template.pptx` 이름으로 이 디렉토리에 저장합니다.
3. 시스템이 자동으로 이 템플릿을 사용합니다.

## 템플릿 요구사항

- 파일 형식: `.pptx`
- 슬라이드 레이아웃:
  - 인덱스 0: 제목 슬라이드
  - 인덱스 1: 제목 + 내용
  - 인덱스 6: 빈 슬라이드

## 참고

템플릿 파일이 없는 경우, python-pptx의 기본 템플릿이 사용됩니다.
