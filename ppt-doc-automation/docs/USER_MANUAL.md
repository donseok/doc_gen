## 1. 시스템 개요

PPT 문서 자동화 시스템은 **마크다운(.md) 파일**을 업로드하거나 직접 입력하여 **PowerPoint 프레젠테이션**을 자동으로 생성하는 웹 애플리케이션입니다.

### 주요 기능
- 📝 마크다운 텍스트로 PPT 자동 생성
- 📂 .md 파일 업로드 지원
- 🎨 PPT 템플릿 선택 가능
- 🌐 다국어 지원 (한국어, 영어, 베트남어)
- 📥 생성된 PPT 즉시 다운로드

---

## 2. 시작하기

### 접속 방법
웹 브라우저에서 아래 주소로 접속합니다:
- **프론트엔드**: http://localhost:5173 (개발 서버)
- **API 문서**: http://localhost:8000/docs

### 언어 변경
화면 우측 상단의 언어 버튼을 클릭하여 언어를 변경할 수 있습니다:
- 🇰🇷 한국어
- 🇺🇸 English
- 🇻🇳 Tiếng Việt

## 3. PPT 생성하기

### 3.1 텍스트 입력 방식

1. 상단 메뉴에서 **"PPT 생성"** 클릭
2. **"텍스트 입력"** 탭 선택
3. 마크다운 에디터에 내용 입력
4. **"🚀 PPT 생성"** 버튼 클릭
5. 생성 완료 후 문서 목록에서 다운로드

#### 입력 예시
```markdown
# 2024년 사업 계획

## 1. 개요
- 연간 매출 목표: 100억원
- 신규 시장 진출 계획

## 2. 전략
- 디지털 전환 가속화
- 글로벌 시장 확대

## 3. Q&A
질문 및 답변
```

### 3.2 파일 업로드 방식

1. 상단 메뉴에서 **"PPT 생성"** 클릭
2. **"파일 업로드"** 탭 선택
3. .md 파일을 드래그 앤 드롭 또는 클릭하여 선택
4. **"🚀 PPT 생성"** 버튼 클릭
5. 생성 완료 후 문서 목록에서 다운로드

> ⚠️ **주의**: .md (마크다운) 파일만 업로드 가능합니다.

## 4. 템플릿 관리

### 템플릿 목록 보기
1. 상단 메뉴에서 **"템플릿"** 클릭
2. 등록된 템플릿 목록 확인
3. 원하는 템플릿의 **"이 템플릿 사용"** 버튼 클릭

### 템플릿 선택 후 PPT 생성
템플릿을 선택하면 자동으로 PPT 생성 페이지로 이동하며, 해당 템플릿이 적용됩니다.

## 5. 문서 관리

### 생성된 문서 확인
1. 상단 메뉴에서 **"문서 목록"** 클릭
2. 생성된 문서들의 상태 확인:
   - ✅ **완료**: 다운로드 가능
   - ⏳ **처리 중**: 생성 진행 중
   - ❌ **실패**: 오류 발생

### 문서 다운로드
1. 상태가 "완료"인 문서의 **"다운로드"** 버튼 클릭
2. .pptx 파일이 다운로드됨

### 문서 삭제
1. 삭제할 문서의 **"삭제"** 버튼 클릭
2. 확인 팝업에서 **"확인"** 클릭

### 상태별 필터링
상단의 필터 버튼으로 문서를 상태별로 필터링할 수 있습니다:
- 전체 / 완료 / 처리 중 / 실패

## 6. 마크다운 작성 가이드

### 기본 구조
```markdown
# 프레젠테이션 제목     ← 표지 슬라이드

## 첫 번째 슬라이드     ← 새 슬라이드 시작
내용 작성

## 두 번째 슬라이드     ← 새 슬라이드 시작
내용 작성
```

### 지원 요소

| 마크다운 | 설명 | PPT 변환 |
|----------|------|----------|
| `# 제목` | 프레젠테이션 제목 | 표지 슬라이드 |
| `## 제목` | 슬라이드 제목 | 새 슬라이드 |
| `### 소제목` | 소제목 | 강조 텍스트 |
| `- 항목` | 불릿 포인트 | 불릿 리스트 |
| `1. 항목` | 번호 목록 | 번호 리스트 |
| ``` 코드 ``` | 코드 블록 | 코드 박스 |

### 완성 예시
```markdown
# 분기별 실적 보고서

## 1. 요약
- 매출 20% 증가
- 신규 고객 500명 획득
- 해외 시장 진출 성공

## 2. 매출 현황

### 국내 매출
- 1분기: 10억
- 2분기: 12억

### 해외 매출
- 1분기: 5억
- 2분기: 8억

## 3. 향후 계획
1. 신제품 출시
2. 마케팅 강화
3. 인력 확충

## 4. Q&A
질문 및 답변 시간
```

## 자주 묻는 질문 (FAQ)

### Q: PPT 생성이 실패합니다
**A**: 마크다운 형식을 확인해주세요. 최소 `# 제목`과 `## 슬라이드` 형식이 필요합니다.

### Q: 이미지를 추가할 수 있나요?
**A**: 현재 버전에서는 로컬 이미지 경로를 통한 이미지 삽입이 지원됩니다.

### Q: 생성된 PPT를 수정할 수 있나요?
**A**: 다운로드 후 PowerPoint에서 자유롭게 수정 가능합니다.
